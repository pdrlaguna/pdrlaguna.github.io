Fill out a form. Get a complete social media kit — posts, stories, carousels, video overlays.

I built a design system that turns content into assets automatically.

Build once, post everywhere.

One podcast episode should become ten pieces of content — quotes for Instagram, audiograms for stories, carousels for LinkedIn. Creating all that by hand takes hours.

I built a system that takes raw content and generates every format automatically.

A design system begins with good design.

The hack: Forms + Figma.

I treated every brand element as a design token. Colors, type styles, image treatments, masks — all components in a Figma library.

Then I built Google Forms that map to content types. A podcast form asks for the script, audio file, and any images. The form feeds data into Figma. Components auto-populate.

One click generates dozens of variations.

A podcast has a script and an audio file: everything goes into a form to be treated behind the curtains.

Sync → Export → Post.

Creating good content is still hard. This system removes the production bottleneck.

One click and you can generate as many variations as you create.

Every new design becomes a template.

Design time now goes toward new content types, not variations. When the brand updates, every asset updates automatically.

Hacked solution, used every week.

Design tools for non-designers.

The system grows with use. Every new template expands what's possible.

Video overlays, Instagram posts, carousels, stories — anyone with access to the form can generate on-brand assets without opening Figma.
